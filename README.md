# 🔍 Chrome Roonyx

Chrome-расширение для блокировки и фильтрации контента в браузере.

## Возможности

- Блокировка нежелательного контента
- Фильтрация по ключевым словам и паттернам
- Настраиваемые правила фильтрации
- Минимальное потребление ресурсов

## Установка

1. Скачайте или клонируйте репозиторий
2. Откройте `chrome://extensions/` в Chrome
3. Включите режим разработчика
4. Нажмите «Загрузить распакованное расширение»
5. Выберите папку репозитория

## Настройка

Настройки доступны через страницу расширения (иконка в панели Chrome).

## Технологии

- **Manifest V3** — современный формат расширений Chrome
- **Vanilla JS** — без зависимостей, быстрая работа
